benefits:
Cache Manager, a free utility from Utilities 4 Dragon, allows you to easily remove configuration files created by various applications. These configuration files are temporary files that store various settings. A variety of circumstances may dictate that these files be reset. Although rarely needed, the program allows you to select which cache files to delete. These configuration files are re-created automatically the next time you run the given program.
- locations supported: Can be used with the old Speech Productivity applications as well as the new Utilities 4 Dragon applications.
- Configuration File Visibility: You can see what any stored setting is set to.
- You Can Delete One Or More Files : You can select which cache files to delete. Easily delete one or all files as needed
